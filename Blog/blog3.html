Apa itu JavaScript? Membuat Web Interaktif

Setelah punya struktur (HTML) dan tampilan (CSS), mungkin kamu ingin websitemu bisa interaktif, misalnya ketika tombol diklik muncul pesan, gambar bisa bergerak, formulir bisa diperiksa sebelum dikirim, dan sebagainya. Nah, di sinilah JavaScript berperan.

Pengertian JavaScript

JavaScript adalah bahasa pemrograman yang berjalan di browser dan memungkinkan kita menambahkan logika serta interaksi pada halaman web. JavaScript dapat membantu menambahkan interaktivitas ke dalam website kita.

Cara Kerja JavaScript
