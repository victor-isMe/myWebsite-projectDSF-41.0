Mengenal apa itu CSS-Sentuhan Kreativitas Pada Web

Setelah mengenal HTML, mungkin kalian merasa "kok webnya kelihatan membosankan" sebab hanya teks hitam di layar browser. Nah, di sinilah peran CSS diperlukan.

Pengertian CSS

CSS singkatan dari Cascading Style Sheets yang fungsinya untuk mengatur tampilan dan gaya pada halaman web HTML kalian. Kalau HTML tadi ibaratnya kerangka fondasi rumah, maka CSS disini berperan sebagai cat dinding, dekorasi, furnitur, dan semua hal yang membuat rumah menjadi lebih enak dilihat.

Cara Kerja CSS

Dalam implementasinya, CSS bisa dipanggil ke dalam sebuah web dengan beberapa cara, inline style(menambahkan CSS langsung ke dalam elemen HTML), internal style(Meletakkan kode CSS di bagian <head> di HTML), dan eksternal style(Memisahkan file dokumen CSS dari file dokumen HTML).
